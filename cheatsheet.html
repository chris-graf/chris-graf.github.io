Ruby Enumerables Cheat Sheet

Enumerators are objects that encapsulate knowledge of how to iterate through a particular collection. Enumerable is a module of methods that can be used on collections, and different datatypes can have their own enumerable methods.

  • Methods that return booleans
    • all?: iterates over collection with a block, if the block returns true for every item, returns true
    • any?: iterates over collection with a block, if the block returns true for at least one item, returns true
    • none? iterates over collection with a block, if the block does not return true for any items, returns true
    • one?: iterates over collection with a block, if the block returns true for exactly one item, returns true
    • include? / member?: checks for existence of an item within the collection, if at least one item equals the argument, returns true
  • map: iterates over a collection with a block and returns a new array after each item has been passed through the block
  • each_with_index: applies a block of code to each item in a collection by its index
  • reduce / inject: applies a binary operation (eg. +) through a block of a symbol to combine all elements of the enumerable
  • select: iterates over a collection, and returns an array of items that return true when passed through the block
  • sort: takes an enumerable and returns them sorted. Can change the order of the sort by passing an optional block.
  • to_a: takes a collection and returns the items in an array
  • to_h: takes a list of two-item collections and returns them as key-value pairs in a hash
